How does it work?

Reveal.js is a web-based framework from presentations


Watch the demo for an introduction

See the project's page for the details

Which text editor?

Any text editor will do the job


But Brackets is a good companion editor

It has a live preview feature to view modifications instantaneously

The interface is highly controllable

Try the key bindings!

Full screen: F / ESC

Slide navigation: Arrows

Speaker's view: S

Slide overview: ESC

Zoom: Alt + Click or Ctrl + Click

The template is highly customisable

  • Change content and structure

  • Include your own logos

  • Develop your own features

Export to pdf

You can export to pdf by adding ?print-pdf to the presentation's URL

path/to/my/pres/index.html?print-pdf path/to/my/pres/index.html?print-pdf&showNotes=true

Or simply click here


Caveat: only works with Chrome
